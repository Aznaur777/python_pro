# Конспекты по Python

Репозиторий создан для конспектов и дз по Python.  
Первая тема: повторение — циклы, массивы (списки), словари.

## Описание
Краткие заметки, примеры кода и задачи для закрепления.

## Тема 1-2 — Повторение
- Циклы: for, while, управление потоком (break, continue, else)
- Массивы (списки): создание, индексация, срезы, методы, списковые включения
- Словари: создание, доступ по ключу, методы, итерирование по ключам/значениям/парам
